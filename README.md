Vortex Front-End

Interface web do projeto **Vortex**, construída com **Next.js**, estilizada com **TailwindCSS** e utilizando componentes modernos do **ReactBits**.  
Este repositório representa a camada visual do Vortex, consumindo a Vortex API.

---

Demonstração
- **Deploy:** https://www.vortexstudio.com.br/
- **API:** https://vortex-api-kb4l.onrender.com

---

Sobre o Projeto

O **Vortex Front-End** foi criado para oferecer uma UI moderna, responsiva e fluida, facilitando a interação do usuário com a API do sistema.  
Este projeto usa:

- Componentização limpa
- Métodos profissionais de organização de páginas
- Requisições otimizadas com Axios/Fetch
- Design consistente usando Tailwind + ReactBits

---

Tecnologias Utilizadas

- **Next.js (App Router)**
- **React**
- **TailwindCSS**
- **ReactBits** (biblioteca de componentes)
- Axios (ou fetch, dependendo do seu uso)
- TypeScript
